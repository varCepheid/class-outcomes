# Class Outcomes

This repository contains projects I've worked on in my computer science college classes.

## Directory

**Adventure Game** is a text-based basic dungeon-crawler game. The first section is a password-guessing game. The second section contains several mini-challenges and a kind of gambling game, with the goal to collect as much treasure from the chest as possible. It is all implemented in one file in Python.

**Egyptian Rat Screw** is a simulated card game. It plays by more or less the rules of the real game with the user's choice of number of players, and the play of the game is recorded in an output text file. It is implemented in Java in six code files plus Javadocs documentation (open `CardGameDocs/index.html` in your browser to see the documentation). There is also a Markdown file that explains the project in greater detail.

**Mario Game** is a simulated game based on the Super Mario Bros. games. Mario moves randomly around a generated world fighting enemies, collecting coins, and advancing through the levels until he reaches the final boss. Each move is recorded in an output text file. It is implement in Java in two code files, and there is a Markdown file that explains the project in greater detail.

The **shell-script projects** folder has a collection of projects created in the bash shell-script language. Each project is in its own folder with a Markdown file explaining it and an input file if necessary.
